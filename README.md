# linQ-chrome-extension

This is a chrome-extension that makes [linQ](https://linq.trap.games/) more useful.

## quick start

1. Run ` npm run build `

2. Set `dist`directory to chrome://extensions/

## install

```sh
npm i
```

## build

```sh
npm run build
```

## lint (format: No)

```sh
npm run lint
```

## lint (format: Yes)

```sh
npm run lint:fix
```
